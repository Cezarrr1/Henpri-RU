# Henpri-RU
Фанатский перевод HENPRI

# О переводе
Сам русификатор является машинным с ручными правками, на основе Grok. Для перевода использовался официальный английский перевод из steam игры. На данный момент переведен весь текст игры и переведён интерфейс игры (небольшая часть интерфейса осталась без перевода).

К игре есть Honorifics Patch, но он практически ничего не меняет. Из него я взял несколько основных моментов и добавил к переводу.

При ручной обработке, я старался исправлять гендерные ошибки, но текста ОЧЕНЬ много, поэтому их осталось думаю немало.

Мат, термины, слэнг старался по возможности правильно адаптировать: игра достаточно специфическая (тематика тюремная), в оригинале очень много пошлости, мата (даже очень много), жесткости и даже жести :) , надеюсь атмосферу смог сохранить. Старался не смягчать и держать на уровне оригинала, и при этом не переборщить.

2. Распакуй в корневую папку с игрой в steam.
3. Играй

## 🤝 Помоги улучшить перевод!
Если будут какие-то ошибки, то напишите. Тестеров у меня нету, сам я только начал играть. Ошибок быть не должно, но мало ли. По коду, по всем проверкам все было норм.

Если есть желание, идеи как можно улучшить перевод, то это можно сделать на гитхабе. Там я выложил исходники текста, там же есть описание как работать с текстом. Если наберется достаточное количество изменений, то можно пересобрать новую версию.

- **Правки в исходники**: Форкни репозиторий → измени файлы в `/src/Text/` → [создай Pull Request](https://github.com/cezarrr1/henpri-ru/compare).
- **Предложения**: Создай [Issue](https://github.com/cezarrr1/henpri-ru/issues/new?template=translation-fix.md).
- **Для новичков**: [Подробная инструкция](how-to-contribute.md).

**Пример PR**: "Исправил 10 фраз в диалогах — лучше звучит!"
