# Как внести свой вклад в русский перевод HENPRI

Спасибо, что хотите помочь сделать перевод лучше!  
Любой вклад — даже исправление одной опечатки — очень ценится.

## Что сейчас нужно больше всего

- Исправление неестественных / корявых фраз  
- Улучшение стиля диалогов (особенно эмоциональных и юмористических моментов)  
- Адаптация шуток и культурных отсылок под русский язык  
- Проверка терминологии (имена, локации, предметы)  
- Исправление грамматики, пунктуации, согласования полов/чисел

## При работе над файлами ВАЖНО:

1. Не добавляйте в файлы символ " иначе это вызывает критическое зависание в игре
2. Следующие символы я проверил, они нормально работают: !#$%&'()*+,\-./:;<=>?@[\]^_`{|}~«»„“”‘’
3. Соблюдайте номера строк, не должно быть смещений по строкам. Если вы изменяете что-то в строке 50, эти изменения должны остаться в строке 50

## Как внести правки (самый простой путь — через GitHub веб-интерфейс)

1. Перейдите в папку с файлами перевода  
   → это `src/text/`

2. Найдите нужный файл  
   Примеры (Все файлы соотносятся друг с другом 1 к 1. Строка в строку.):  
   - src\Text\TextRu\5.txt - текст на русском
   - src\Text\TextEn\5.txt - текст оригинала
   - src\Text\NamesRu\n5.txt - имена на русском
   - src\Text\NamesEn\n5.txt - имена оригинала

3. Нажмите на файл → кнопку **карандаш** (Edit this file)

4. Внесите изменения прямо в браузере

5. В самом низу страницы заполните:
   - Commit message (на русском или английском):  
     примеры хороших сообщений:
     - «Улучшил диалоги Софии в файле 5 — звучит естественнее»
     - «Исправил опечатку там-то»
     - «Перефразировал шутку, пун там-то»

   - (опционально) добавьте расширенное описание

6. Нажмите **Propose changes** → **Create pull request**

Готово! Ваш Pull Request появится — автор(ы) его увидят и рассмотрят.

## Если хотите работать локально (через Git)

```bash
# 1. Форкните репозиторий (кнопка Fork вверху справа)

# 2. Склонируйте свой форк
git clone https://github.com/ВАШ-ЛОГИН/название-репозитория.git
cd название-репозитория

# 3. Создайте ветку для своих правок (желательно осмысленное имя)
git checkout -b fix-sophia-dialogs
# или
git checkout -b improve-ui-terms

# 4. Внесите изменения в файлы

# 5. Зафиксируйте изменения
git add .
git commit -m "Улучшены диалоги Софии в сцене такой-то"

# 6. Отправьте ветку в ваш форк
git push origin fix-sophia-dialogs

# 7. Перейдите на страницу вашего форка в браузере
# → появится зелёная кнопка "Compare & pull request" → создайте PR
